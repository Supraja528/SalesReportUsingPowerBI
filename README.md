# 📊 Power BI Dashboard – Boxes Shipped Analysis

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze and visualize **Boxes Shipped** across different dimensions. The dashboard provides valuable insights into sales performance, country-wise distribution, and shipment trends over time, enabling better data-driven decision-making.

---

## 🎯 Objectives

- Analyze shipment performance by sales representatives.
- Compare shipment volumes across different countries.
- Track shipment trends over time.
- Visualize business performance using interactive charts and reports.

---

## 📂 Dataset

The dataset contains shipment-related information, including:

- Date
- Sales Person
- Country
- Boxes Shipped

---

## 📈 Dashboard Features

### 📌 Sales Person Performance
- Displays total boxes shipped by each sales representative.
- Helps identify top-performing sales personnel.

### 🌍 Country-wise Analysis
- Compares shipment volumes across different countries.
- Highlights geographical distribution of shipments.

### 📅 Time Series Analysis
- Shows shipment trends over time using a line chart.
- Helps identify seasonal or monthly shipment patterns.

### 📊 Waterfall Analysis
- Visualizes cumulative shipment changes over different periods.
- Helps understand business growth and shipment contribution.
